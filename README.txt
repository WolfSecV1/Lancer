A lightweight paint application built using C++ and Qt 6.

## Features
- Mouse drawing
- Color swatch picker
- Clear canvas button
- Clean and minimal UI

## Build Instructions

1. Open `ProLancer.sln` in Visual Studio 2022
2. Make sure Qt is configured under `Qt Project Settings`
3. Build in **Release** mode
4. Run `windeployqt ProLancer.exe` to bundle dependencies
